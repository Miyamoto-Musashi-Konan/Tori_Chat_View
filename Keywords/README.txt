    └── visualizer.html    ← 시각화 앱 (브라우저에서 열기)

[categories.txt 수정 방법]

1. 새 태그 추가:
   해당 SUBCATEGORY의 TAGS 줄 끝에 쉼표 후 추가
   예) TAGS: 기존태그1, 기존태그2, 새태그

2. 새 서브카테고리 추가:
   해당 CATEGORY 블록 안에 아래 형식으로 추가
   SUBCATEGORY: 영문ID | 한글명
     TAGS: 태그1, 태그2, 태그3

3. 새 카테고리 추가:
   파일 끝 "END OF TAXONOMY" 위에 아래 형식으로 추가
   CATEGORY: 영문ID | 한글명
     SUBCATEGORY: 영문ID | 한글명
       TAGS: 태그1, 태그2

4. 주석 처리: # 기호로 시작하면 무시됨
